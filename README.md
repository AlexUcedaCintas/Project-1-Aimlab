# AIM LAB
    This game is an Aim trainer
    You can improve your speed and time reaction.

## HOW TO PLAY
    -Select the quantity of targets
    -Select the Difficulty (speed of the spawn and dispawn of objectives)
    -Select a pointer 
    -Hit as many balls as you can
